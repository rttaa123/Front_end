## 1. 响应式设计实现方式

### 1.1 视口（Viewport）设置
HTML 中通过：

```html
<meta name="viewport" content="width=device-width, initial-scale=1.0, minimum-scale=1.0, maximum-scale=1.0, user-scalable=no">
```

确保移动端以设备宽度进行布局，并限制缩放。

### 1.2 布局自适应（宽度/弹性布局）
- 页面外层容器 `.Warp`：
  - `width: 100%` + `max-width: 1720px`，在大屏居中显示，小屏可收缩。
- 顶部 `.header`、导航区 `.nav` 等多处使用 `display: flex` 完成横向/纵向弹性布局。
- 多处使用 `calc()`（如 `.header .nav { width: calc(100% - 830px); }`）实现随容器变化的自适应宽度。

### 1.3 媒体查询（@media）
我在 CSS 末尾提供了基础的小屏适配：

```css
@media (max-width: 1367px) {
  body { padding: 0; }
  .Warp { width: 100%; padding: 15px; box-sizing: border-box; }
}
```

在屏幕较小时减少 body 的上下留白，并给整体容器增加内边距，避免贴边。

### 1.4 JS 动态缩放（zoom）
`main.js` 中在页面加载与窗口变化时，根据窗口宽度对整体 `.Warp` 做缩放：

- 计算比例：`bl = windowWidth / 1920`
- 当 `windowWidth <= 1920` 时：`$('.Warp').css('zoom', bl)`

这是一种“整体缩放式”响应，让我的设计在不同分辨率下按比例缩放。

## 2. 已实现的 CSS 动画 / 交互效果（Hover/Transition）

> 本次作业动效主要通过 `transition + transform/filter/opacity/box-shadow` 来实现。

### 2.1 Logo 悬停 3D 旋转
- `.header .logo` 设置 `perspective: 1000px`
- `img` 设置 `transform-style: preserve-3d`、`transition`
- 悬停时：`transform: rotateX(360deg)`

### 2.2 导航按钮交互（颜色切换 + 背景上浮）
- 图标双图层（`.icon-light` / `.icon-dark`）通过 `opacity` 过渡切换。
- `a::before` 背景高度从 `0 -> 100%` 形成“从底部浮现”的按钮填充效果。

### 2.3 顶部右侧小图标悬停（缩放+旋转）
`.header .nav dl dd a:hover`：
- `transform: scale(1.1) rotate(5deg)`
- `opacity` 降低


### 2.4 Layer1 底部三张图悬停放大
`.indexContainer .layer1 ul li:hover img`：
- `transform: scale(1.1)`
- `opacity: 0.9`

### 2.5 Layer2（订阅区）整体与组件悬停效果
- 整块悬停：阴影发光（橙色 glow）
  - `.layer2:hover { box-shadow: 0 0 30px rgba(255, 165, 0, 0.6); }`
- 标题图片悬停：缩放
- 文字悬停：`text-shadow` 发光
- 输入框 hover/focus：`box-shadow` 淡绿色发光
- 按钮 hover：上移 + 阴影 + 变色

### 2.7 Layer4 区域动效

## 3. JS 实现的交互功能（非 CSS 动画）
>虽然本次作业不要求JS效果，但是在老师讲课之后我添加到了作业中，故也简单介绍
### 3.1 轮播图（Carousel）
- 自动播放：`setInterval` 每 3 秒切换
- 左右箭头点击切换上一张/下一张
- 指示点点击切换到指定页
- 鼠标悬停暂停，移出继续播放

### 3.2 轮播组件相关动效（CSS + JS）
- CSS：轮播项 `.carousel-item` 使用 `opacity` 的过渡实现淡入淡出：
  - `transition: opacity 2s ease-in-out`
- CSS：轮播容器悬停时，当前图片轻微放大：
  - `.carousel-container:hover .carousel-item.active img { transform: scale(1.02); }`


### 3.2 导航激活态管理
- 根据 URL hash（如 `#home/#about`）设置对应导航项 `active`
- 点击导航：切换 `active`，更新 hash，并滚动到顶部
